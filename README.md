Socail spark is a app that helps users mantain social connections with their loved ones by suggesting small, meaningful sparks depending on the time of thenday.
The goal of this project is for it to give simple social interaction suggestions.
This project consits of the following features; socail suggestions based on user input, easy to use UI.
Socail suggestions are:
morning - Send a "Good Morning" text 
mid morning - Thank a colleague
afternoon - Share a meme
snack tim -Send a "thinking of you" message
dinner - call a friend
night - comment on a friends post.

The language used to code: Kotlin
IDE: Andriod studio
